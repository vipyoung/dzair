QARTA: AI Enabled Map Engine

QARTA is a free an unlimited map engine powered by open source products such as OSM, OSRM, and Nominatim, as well as state of the art AI technology developed @QCRI.

Turn-by-Turn Navigation

Turn-by-turn navigation also known as "direction" is a service that allows you find a navigation route between any two points on earch: Origin and destination. Currently, the API expects origin and destination to be provided as pairs of (latitude,longitude). The underlying service uses an optimized OSRM server.

Syntax

Currently, the API uses the exact same syntax of OSRM.

Example

http://qarta.qcri.org/route/v1/driving/51.61554,25.26004;51.61464,25.25959?steps=true
